CH4: Expected Value (Mean) of X

Discrete: μ = E(X) = Σx x·f(x) Continuous: μ = E(X) = ∫−∞∞ x·f(x) dx Probability-weighted average of all possible values of X

Expected Value of g(X)

Discrete: E[g(X)] = Σx g(x)·f(x) Continuous: E[g(X)] = ∫−∞∞ g(x)·f(x) dx Replace x·f(x) with g(x)·f(x) — same probability function f(x)! E(aX+b) = a·E(X) + b  |  E(X±Y) = E(X)±E(Y) (always)

Expected Value of g(X,Y) — Joint

Discrete: E[g(X,Y)] = ΣxΣy g(x,y)·f(x,y) Continuous: E[g(X,Y)] = ∫∫ g(x,y)·f(x,y) dx dy If independent: E(XY) = E(X)·E(Y)  |  f(x,y)=g(x)h(y) ⇒ independent

CH4: Variance of X

σ² = Var(X) = E[(X−μ)²] Shortcut: Var(X) = E(X²) − μ² Discrete: σ² = Σ(x−μ)² f(x) Continuous: σ² = ∫(x−μ)² f(x)dx σ = √σ²  (standard deviation)

Variance of g(X) — Other Functions

σ²g(X) = E{[g(X) − μg(X)]²} Discrete: Var[g(X)] = Σ[g(x)−μg]² f(x) Continuous: Var[g(X)] = ∫[g(x)−μg]² f(x)dx Find μg first via E[g(X)], then compute variance

CH4: Variance Properties (CRITICAL)

  • Var(aX + b) = a²·Var(X)  — constant b has NO effect
  • Var(3X + 5) = 9·Var(X)  ← not 3, not 14
  • Multiply by a → variance × a²
  • Add constant → variance unchanged

Linear Combos: aX + bY + c

Var(aX+bY+c) = a²σ²X + b²σ²Y + 2ab·σXY Var(aX−bY+c) = a²σ²X + b²σ²Y − 2ab·σXY Var(X+Y) = σ²X + σ²Y + 2·Cov(XY) Var(X−Y) = σ²X + σ²Y − 2·Cov(XY)
  • X,Y independent: Cov=0 → Var(X±Y)=σ²X+σ²Y
  • X,Y non-linear transforms change distribution shape

Expected Value Rules

E(aX + b) = a·E(X) + b E(X ± Y) = E(X) ± E(Y)  (always) E(XY) = E(X)·E(Y)  only if independent

CH4: Covariance

σXY = E[(X−μX)(Y−μY)] Shortcut: σXY = E(XY) − μX·μY Discrete: ΣxΣy(x−μX)(y−μY)f(x,y) Continuous: ∫∫(x−μX)(y−μY)f(x,y)dx dy
  • σXY > 0 → X & Y move together
  • σXY < 0 → X & Y move inversely
  • σXY ≈ 0 → weak linear relationship
  • Independent → σXY=0, but NOT the reverse!

CH4: Correlation Coefficient

ρXY = σXY / (σX · σY) −1 ≤ ρXY ≤ 1
  • Scale-free measure of linear association
  • Measures strength AND direction
  • ρ=0 when X,Y independent (converse not always true)
  • Covariance = direction only; Correlation = direction + strength

CH4: Chebyshev's Theorem

P(μ−kσ < X < μ+kσ) ≥ 1 − 1/k²
kAt least…
275% within 2σ
388.89% within 3σ
493.75% within 4σ
  • Works for ANY distribution (no shape assumption)
  • Sets a weak lower bound — actual % is usually higher
  • Normal distribution gives tighter results (68-95-99.7)

Steps for Chebyshev problems

1. Find σ=√σ²   2. Distance from mean = |bound−μ|   3. k=distance/σ   4. P ≥ 1−1/k² Example: μ=8, σ²=9(σ=3), find P(−4<X<20): distance=12, k=12/3=4, P≥1−1/16=93.75%

CH5: Binomial Distribution

b(x; n, p) = C(n,x) · px · qn−x    x=0,1,...,n; q=1−p E(X) = np    Var(X) = npq    Skewness = (q−p)/√(npq)
  • n = # trials; p = P(success); q = P(failure) = 1−p
  • p=0.5 → symmetric | p<0.5 → right skew | p>0.5 → left skew
  • Notation: X~Bin(n,p)

Reading the CDF Table (Table A.1)

  • Table gives P(X ≤ r) for given n and p
  • P(X < 10) = P(X ≤ 9)  ← use r=9, not 10
  • P(4 ≤ X ≤ 7) = P(X≤7) − P(X≤3)
  • P(X ≥ 6) = 1 − P(X≤5)

Excel

=BINOM.DIST(x, n, p, FALSE) → exact P(X=x) =BINOM.DIST(x, n, p, TRUE) → cumulative P(X≤x) P(4≤X≤7): =BINOM.DIST(7,n,p,TRUE)−BINOM.DIST(3,n,p,TRUE) Probabilities do NOT scale with n! Recalculate each problem.

CH5: Multinomial Distribution

f = [n! / (x₁!x₂!···xₖ!)] · p₁x₁p₂x₂···pₖxₖ E(Xᵢ) = n·pᵢ    Var(Xᵢ) = n·pᵢ·qᵢ
  • Generalization of binomial to k > 2 outcomes
  • Requires: Σxᵢ = n  and  Σpᵢ = 1
  • Scale dependent — must recalculate for different n
Excel: =MULTINOMIAL(x₁,x₂,...)·p₁^x₁·p₂^x₂·...  (multiply separately)

CH5: Negative Binomial

P(X=x) = C(x−1, k−1) · pk · qx−k    x=k, k+1, ... E(X) = pr/(1−p)    Var(X) = pr/(1−p)²    (r=x−k)
  • k = number of successes needed; x = trial where k-th success occurs
  • Last trial is always a success; arrange k−1 successes in x−1 prior trials
Excel: =NEGBINOM.DIST(r, k, p, FALSE) where r = x−k = # failures before k-th success Example: 5th success on 7th trial, p=0.6: P(X=7)=C(6,4)(0.6)⁵(0.4)²=0.1866

CH5: Geometric Distribution

g(x; p) = p · qx−1    x = 1, 2, 3, ... E(X) = 1/p    Var(X) = (1−p)/p²
  • Special case of Neg. Binomial with k=1
  • X = trial number of the first success
  • Always right-skewed (long right tail)
Excel: =NEGBINOM.DIST(x−1, 1, p, FALSE)  ← r=x−1 failures before 1st success Example p=0.05, P(X=5) = (0.05)(0.95)⁴ = 0.0407  |  E(X)=1/0.05=20

CH5: Poisson Distribution

p(x; λt) = e−λt(λt)x / x!    x = 0, 1, 2, ... E(X) = Var(X) = λt    (mean = variance always!) Skewness = (λt)−½
  • λ = avg rate per unit time/distance/area
  • t = specific time/region of interest
  • Events are independent; no memory
  • Larger λt → more symmetric distribution

Changing the Time Interval

If λ=4/ms and you want 2ms window → new λt = 4×2 = 8

Using Poisson CDF Table (Table A.2)

Table gives P(X≤r) for given μ=λt P(X=6) = P(X≤6) − P(X≤5)  (subtract consecutive rows) Example: λt=4, P(X=6) = 0.8893 − 0.7851 = 0.1042

Excel

=POISSON.DIST(x, λt, FALSE) → exact P(X=x) =POISSON.DIST(x, λt, TRUE) → cumulative P(X≤x)

CH5: Poisson Approximation to Binomial

Use when: n > 20 AND np < 5 (or nq < 5) Set μ = λt = np, then use Poisson formula Rare events, many trials → Poisson is easier and equally accurate Example: p=0.005, n=400 → np=2, use Poisson with λt=2

Skewness = (q−p)/√(npq)

  • p < 0.5 → right-skewed (positive skewness)
  • p > 0.5 → left-skewed (negative skewness)
  • Larger n with p near 0.5 → more symmetric

CH6: Uniform Distribution

X~Unif(A,B):   f(x) = 1/(B−A),   A≤x≤B E(X) = (A+B)/2 Var(X) = (B−A)²/12 P(a≤X≤b) = (b−a)·f(x) = (b−a)/(B−A)
  • Rectangle: base = B−A, height = 1/(B−A), area = 1
  • Probability = area of rectangle = base × height
  • Symmetric distribution

Example: X~Unif(1,4)

f(x) = 1/3  |  E(X) = 5/2 = 2.5  |  Var(X) = 9/12 = 3/4 P(X≤3) = (3−1)·(1/3) = 2/3 = 0.667

Computing E(X) and Var(X) from scratch (integral)

E(X) = ∫AB x·[1/(B−A)] dx = (B²−A²)/[2(B−A)] = (A+B)/2 E(X²) = (B³−A³)/[3(B−A)] = (B²+AB+A²)/3 Var(X) = E(X²) − μ² = (B−A)²/12

CH6: Normal Distribution

X~N(μ,σ): f(x) = (1/√(2πσ)) · e−(x−μ)²/2σ² E(X) = μ    Var(X) = σ²
  • Bell-shaped, symmetric about μ
  • Mean = Median = Mode (all at center μ)
  • Inflection points at μ ± σ
  • Approaches x-axis asymptotically (never touches)
  • Total area under curve = 1
  • Characterized by exactly two parameters: μ and σ

Empirical Rule (68-95-99.7)

RangeApprox. %
μ ± 1σ68%
μ ± 2σ95%
μ ± 3σ99.7%

Two Normals with Different μ: same σ → same shape, shifted

Two Normals with Different σ: wider σ → flatter, lower peak; smaller σ → taller, narrower

CH6: Standard Normal Distribution

Z~N(0,1): mean=0, std dev=1 z = (x − μ) / σ  ← standardize any normal to Z
  • z-score = # of standard deviations x is from μ
  • z > 0: x is above mean  |  z < 0: x is below mean
  • Standard Normal table gives P(Z < z) — area to the LEFT

CH6: Using the Z-Table

Forward: x → probability

1. Compute z = (x−μ)/σ   2. Look up z in table → get P(Z<z)   3. Adjust for direction

Reverse: probability → x

x = μ + z·σ 1. Find desired probability in table body   2. Read off z   3. Apply x = μ + z·σ

CH6: Normal Approx. to Binomial

Condition: np ≥ 5 AND nq ≥ 5 Use Normal with: μ=np  |  σ=√(npq)
  • Works best in center of bell curve
  • Better approximation when p near 0.5 and n large
  • Use Z-table after standardizing

P(X≤x) ≈ P(Z ≤ (x+0.5−np)/√(npq))

Critical Traps & Key Facts

Chapter 4

  • Var(aX+b) = a²Var(X) — adding constant does NOTHING
  • Var(3X+5) = 9Var(X), NOT 14Var(X)
  • Cov(X,Y)=0 does NOT always mean independent
  • Independent DOES mean Cov=0 (one direction only)

Chapter 5

  • P(X<10) ≠ P(X≤10) for discrete RVs!
  • P(X<10) → r=9 in CDF table
  • Poisson: mean always equals variance
  • Geometric: X starts at 1, not 0
  • Neg. Binomial: last trial is always the k-th success

Chapter 6

  • Z-table gives area to the LEFT always
  • P(X=any value) = 0 for continuous RVs
  • Symmetry shortcut: P(X<μ) = 0.5 always
  • P(Z>z) = 1 − P(Z<z)
  • Continuity correction: discrete ± 0.5 → continuous range

Problem-Solving Steps

Normal probability problem

1. Identify μ and σ   2. Draw picture!   3. Shade region   4. Convert to z: z=(x−μ)/σ   5. Use table/Excel   6. Adjust for direction (1− if right tail)

Binomial probability

1. Identify n, p, q=1−p   2. Use formula, table (A.1), or Excel   3. Watch: < vs ≤ for discrete!

Poisson probability

1. Identify λ and t (compute λt)   2. Adjust t if interval changes   3. Use formula, table (A.2), or Excel   4. For range: P(X=x) = P(X≤x)−P(X≤x−1)

Chebyshev's Theorem

1. Find σ   2. k = distance from μ / σ   3. P ≥ 1−1/k²
